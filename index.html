Sobre este proyecto

Este proyecto es un estudio crítico y comparado de los textos sagrados de las principales tradiciones religiosas del mundo. No pretende ser apologético ni anticlerical: su objetivo es que cualquier persona pueda leer, con contexto académico, lo que dicen realmente estos textos.

Todo el contenido está extraído de fuentes de dominio público o de repositorios académicos reconocidos. Cada tarjeta enlaza directamente al texto original para que puedas verificar cualquier afirmación.

Principios editoriales

Cada tarjeta distingue entre (1) lo que dice el texto, (2) el contexto histórico en que fue escrito, y (3) cómo lo interpreta la tradición académica contemporánea. Las notas críticas no son opinión del autor: reflejan el debate académico documentado.

Qué no es este proyecto

No es un ataque a ninguna fe ni una defensa de ninguna. No pretende determinar qué es verdad religiosamente. Es una herramienta para leer los textos con información contextual que normalmente no está disponible para el lector no especializado.

Transparencia total

Cada afirmación que se hace sobre un texto tiene un enlace directo al fragmento original. Si crees que algo está mal interpretado o sacado de contexto, el enlace te permite comprobarlo. El debate informado es bienvenido.

Código abierto

Este proyecto es de código abierto y está publicado en GitHub. Cualquiera puede revisar su código, proponer correcciones o añadir nuevos textos siguiendo los mismos estándares editoriales.
